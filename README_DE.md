## 🚀 Schnellstart

<details>
<summary>📖 <b>Klicken zum Erweitern</b></summary>

### ⚡ Installation in 3 Schritten

```bash
# Schritt 1: Repository klonen
git clone https://github.com/vantisCorp/V-Sentinel.git

# Schritt 2: Abhängigkeiten installieren
cd V-Sentinel
cargo install --path .

# Schritt 3: V-Sentinel starten
sentinel start --mode production
```

### 🔧 Konfiguration

```toml
# sentinel_config.toml
[core]
mode = "production"
log_level = "info"

[ai]
model = "gpt-4"
threshold = 0.95

[gaming]
anti_ddos = true
ram_defolding = true
```

</details>

## 🛠️ Systemanforderungen

### Minimum
- **OS**: Linux (Ubuntu 20.04+, Debian 11+), Windows 10+, macOS 11+
- **RAM**: 8 GB
- **CPU**: 4 Kerne
- **Festplatte**: 10 GB
- **Rust**: 1.70+

### Empfohlen
- **OS**: Linux (Ubuntu 22.04+)
- **RAM**: 16 GB+
- **CPU**: 8 Kerne+
- **Festplatte**: SSD 50 GB+
- **GPU**: NVIDIA RTX 3060+ (für KI-Beschleunigung)
