## 🚀 快速开始

<details>
<summary>📖 <b>点击展开</b></summary>

### ⚡ 3步安装

```bash
# 步骤1：克隆仓库
git clone https://github.com/vantisCorp/V-Sentinel.git

# 步骤2：安装依赖
cd V-Sentinel
cargo install --path .

# 步骤3：运行V-Sentinel
sentinel start --mode production
```

### 🔧 配置

```toml
# sentinel_config.toml
[core]
mode = "production"
log_level = "info"

[ai]
model = "gpt-4"
threshold = 0.95

[gaming]
anti_ddos = true
ram_defolding = true
```

</details>

## 🛠️ 系统要求

### 最低要求
- **操作系统**：Linux（Ubuntu 20.04+、Debian 11+）、Windows 10+、macOS 11+
- **内存**：8 GB
- **CPU**：4核
- **磁盘**：10 GB
- **Rust**：1.70+

### 推荐配置
- **操作系统**：Linux（Ubuntu 22.04+）
- **内存**：16 GB+
- **CPU**：8核+
- **磁盘**：SSD 50 GB+
- **GPU**：NVIDIA RTX 3060+（用于AI加速）
